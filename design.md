# Gym Tracker - Visual Design Style

## Design Philosophy

### Color Palette
- **Primary**: Warm Terracotta (#D2691E) - energetic yet sophisticated
- **Secondary**: Sage Green (#9CAF88) - calming and natural
- **Accent**: Cream (#F5F5DC) - clean and minimal
- **Neutral**: Charcoal (#36454F) - professional text color
- **Background**: Off-white (#FEFEFE) - clean canvas

### Typography
- **Display Font**: "Playfair Display" - elegant serif for headings and hero text
- **Body Font**: "Inter" - clean, modern sans-serif for readability
- **Accent Font**: "JetBrains Mono" - monospace for data and metrics

### Visual Language
- **Minimalist Aesthetic**: Clean lines, generous white space, purposeful design
- **Organic Shapes**: Subtle curves and flowing elements that suggest movement
- **Sophisticated Textures**: Subtle gradients and depth without overwhelming
- **Editorial Inspiration**: Drawing from high-end fitness and wellness publications

## Visual Effects & Styling

### Used Libraries
- **Anime.js**: Smooth micro-interactions and button animations
- **ECharts.js**: Clean, minimal data visualizations with custom color schemes
- **Splide.js**: Elegant image carousels for progress galleries
- **p5.js**: Subtle background particle effects and organic animations
- **Pixi.js**: Smooth visual transitions and hover effects

### Animation & Effects
- **Scroll Animations**: Gentle fade-ins with subtle upward motion (16px max)
- **Hover States**: 3D tilt effects on cards, color morphing on buttons
- **Loading States**: Organic pulse animations with terracotta accents
- **Data Transitions**: Smooth chart animations with staggered reveals

### Header Effects
- **Hero Background**: Generated abstract fitness imagery with geometric flows
- **Text Effects**: Gradient color cycling on key metrics and achievements
- **Interactive Elements**: Subtle shadow expansion and color shifts

### Layout Principles
- **Grid System**: 12-column responsive grid with consistent spacing
- **Visual Hierarchy**: Clear information architecture with purposeful sizing
- **Mobile-First**: Touch-friendly interactions with generous tap targets
- **Content Flow**: Logical reading patterns and intuitive navigation

### Component Styling
- **Cards**: Soft shadows with rounded corners, hover lift effects
- **Buttons**: Solid terracotta with white text, subtle scale on hover
- **Forms**: Clean inputs with sage green focus states
- **Charts**: Muted color palette with terracotta accent highlights

### Background Treatment
- **Consistent Base**: Off-white background throughout all pages
- **Decorative Elements**: Subtle geometric shapes and organic patterns
- **Section Differentiation**: Slight tonal variations using sage green tints
- **Visual Interest**: Abstract generated imagery for hero sections
